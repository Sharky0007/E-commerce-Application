# E-commerce-Application
This is an e-commerce application , which has a feature to manage user, manage products,add products to cart and also has the facility to make payments.

# Features
User registration, authentication, and profile management.
​Product catalog management, including create, update, view, and delete products.
​Shopping cart functionality to add, update, and remove items.
​Payment flow to place orders and complete purchases.
​

# Pre-requisite
Java 17 (or your project’s Java version).
​Spring Boot (Web, Data JPA, Security where applicable).
​Maven for build and dependency management.
​Relational database (e.g., MySQL/PostgreSQL/H2) for persistence.
